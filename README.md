# DRONE_PROJECT



Description
This repository contains the complete design, code, and documentation for an IoT and AI-enabled Rescue Drone prototype. The system is designed to assist rescue teams during natural disasters by providing a fast, low-cost aerial response for detecting stranded individuals and transmitting their real-time GPS coordinates.
